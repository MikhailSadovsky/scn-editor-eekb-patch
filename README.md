# scn-editor-eekb-patch

Applying updates for SCn editor to make expert mode working for EEKB.
Usage:
* ```git clone https://github.com/MikhailSadovsky/scn-editor-eekb-patch.git```
* ```cd scn-editor-eekb-patch```
* ```./apply_patch.sh```

How to disable SCn editor:
* ```cd sc-web```
* ```grunt build```
